機能一覧

Features

オーバーエンチャント検知

Over-Enchantment Detection

不正なエンチャントレベルを持つアイテムを自動的に検出します。

Automatically detects items with illegal enchantment levels.

KillAura検知

KillAura Detection

不自然な攻撃パターンを検出し、KillAuraなどのチートを防止します。

Detects unnatural attack patterns to prevent KillAura and similar cheats.

TPS測定

TPS Measurement

サーバーのTPS（Ticks Per Second）をリアルタイムで測定し表示します。

Measures and displays server TPS (Ticks Per Second) in real-time.

ユーザーデータ表示

User Data Display

プレイヤーの詳細情報を簡単に確認できます。

Easily check detailed information about players.

インベントリ閲覧

Inventory Viewing

管理者がプレイヤーのインベントリをリアルタイムで確認できます。

Administrators can check player inventories in real-time.

権限設定

Permission Settings

詳細な権限システムで、誰が何を使えるかを細かく設定できます。

Detailed permission system allows fine control over who can use what.

禁止アイテム

Banned Items

特定のアイテムの使用や所持を禁止できます。

Ban the use or possession of specific items.

禁止エンティティ

Banned Entities

特定のエンティティのスポーンや使用を制限できます。

Restrict the spawning or use of specific entities.

エンティティリスト閲覧

Entity List Viewing

ワールド内のすべてのエンティティを一覧表示します。

List all entities in the world.

アイテムキル

Item Kill

ドロップしているアイテムを一括で削除できます。

Remove all dropped items at once.

その他多数の機能

Many More Features

常に新機能を追加しています。GitHubで最新の更新をチェックしてください。

We're constantly adding new features. Check GitHub for the latest updates.
